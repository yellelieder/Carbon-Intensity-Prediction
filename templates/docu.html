Technical Documentation

General

The system is written in python. The frontend is only for demo puroses and basically plain html. There are a few JavaScript functions and a tiny fraction of css

Server

The webserver uses Flask and Flask Restfull.

Scraper

To predict future energy production and consumption, historial data are scraped from smard.de. Most recent data is automatically scraped by the system to further improve the prediction accuracy.

Data cleaning

Data cleaning is automatically done by the system after cleaning. All the input csv files are always merged and prepared to perform machine learining on them.

Weather API

openweathermap.org provide free access for student to some of their APIs. The system conciders a 30 day, daily climate forcast and a 4 day hourly weather forcast

Machine Learning

Predictions are based on a Auto Regression Model from the python statsmodels library.

The prediction made with the AR Model has a RMSE of about 1404 MWh.
